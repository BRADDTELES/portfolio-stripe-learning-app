Stripe Learning App

Um aplicativo de aprendizado construído com Kotlin Multiplatform e Compose, explorando o ecossistema de pagamentos da Stripe de ponta a ponta.

Sobre o Projeto

A Jornada de Aprendizado

Este projeto nasceu como um desafio de aprendizado: construir um sistema completo, desde o backend até o aplicativo móvel, para dominar a integração de pagamentos com a Stripe em um ambiente moderno.

Utilizando Kotlin Multiplatform, a lógica de negócio, acesso a dados e ViewModels são compartilhados, enquanto a UI é construída com Compose Multiplatform, rodando nativamente no Android.

Arquitetura Robusta

A arquitetura segue os princípios de Clean Architecture + MVVM. O sistema é dividido em duas partes: o App Cliente e um Backend Ktor seguro, que protege as chaves da API e orquestra as chamadas sensíveis, garantindo um fluxo de pagamento seguro e escalável.

Funcionalidades

Gestão de Produtos

CRUD completo para criar, editar e remover produtos no catálogo do app.

Pagamento Seguro

Fluxo de pagamento com cartão utilizando o PaymentSheet nativo da Stripe para Android.

Controle de Estoque

Sistema para gerenciar a quantidade de produtos e registrar movimentações de entrada e saída.

Histórico de Compras

Acompanhe todas as transações em um histórico detalhado, com filtros e estatísticas.

Comprovantes e Boleto

Geração de comprovantes em PDF e visualizações de Boletos, podendo compartilhar os arquivos.
